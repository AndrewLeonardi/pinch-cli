# Security Policy

## Supported Versions

| Version | Supported |
| ------- | --------- |
| 1.x     | Yes       |
| < 1.0   | No        |

## Reporting a Vulnerability

Please **DO NOT** open a public issue for security vulnerabilities.

Email **security@pinchers.ai** with:
- Description of the vulnerability
- Steps to reproduce
- Potential impact

We will respond within 48 hours and work with you on a fix before any public disclosure.

## Scope

This policy covers the `pinch-cli` npm package and its generated project scaffolding. Issues with Cloudflare Workers, Supabase, or other third-party services should be reported to those providers directly.
